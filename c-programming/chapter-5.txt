RECURTION
--------------------- 

when a function calls itself its called recursion

PROPERTY OF RECURSION
----------------------

a. Anything that can be done with iteration, can be done with recursion and vice -versa 
b. Base Case if the condition which stop recursion
c. Iteration has infinite loop & Recursion has stack over flow 
d. Recursion can sometimes give the most simple solution
