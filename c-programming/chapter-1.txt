variable 
-------------------------

 DEF :-   variable is a name of memory location where we store some data .

 RULE :- 
        1. variable are case sensitive 
        2. 1st character is alphabet or '  _  '
        3. no comma / blank space 
        4. No other symbol other that '  _  '

NOTE :- :-       variable can be updated .
                variable name should be meaningfull
